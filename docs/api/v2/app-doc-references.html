app.doc.references

문서 안의 책갈피, 하이퍼링크, 숨은 설명, 메모 이동을 관리합니다.

책갈피

API설명연동 Action
bookmarks.add(name, options)현재 위치에 책갈피를 추가합니다.Bookmark Action + BookMark ParameterSet
bookmarks.moveTo(name, options)지정한 책갈피로 이동합니다.Bookmark Action + BookMark ParameterSet
bookmarks.modify(name, newName, options)기존 책갈피로 이동한 뒤 새 이름으로 변경합니다.Bookmark, ModifyBookmark Action + BookMark ParameterSet
bookmarks.dialog.open()책갈피 편집 대화상자를 엽니다.BookmarkEditDialog Action

하이퍼링크

API설명연동 Action
hyperlinks.insert(options)현재 위치 또는 선택 영역에 하이퍼링크를 삽입합니다.Hyperlink Action + HyperLink ParameterSet
hyperlinks.jump(options)하이퍼링크 대상으로 이동합니다.HyperlinkJump Action + HyperlinkJump ParameterSet
hyperlinks.next()다음 하이퍼링크로 이동합니다.HyperlinkForward Action
hyperlinks.previous()이전 하이퍼링크로 이동합니다.HyperlinkBackward Action

숨은 설명

API설명연동 Action
comments.insert()숨은 설명을 삽입합니다.Comment Action
comments.modify()숨은 설명을 수정합니다.CommentModify Action
comments.delete()숨은 설명을 삭제합니다.CommentDelete Action

메모

API설명연동 Action
memos.next()다음 메모로 이동합니다.MemoToNext Action
memos.previous()이전 메모로 이동합니다.MemoToPrev Action
