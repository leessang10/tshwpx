TSHWPX API 구조

app은 실행 중인 한글 자동화 세션이고, app.doc은 그 세션의 현재 문서 작업 진입점입니다.

모듈 목록

모듈명역할하위 모듈명
app 실행 중인 한글 프로세스/자동화 세션 -
app.doc 현재 활성 문서 작업 진입점
문서 모듈 보기
  • app.doc.text 텍스트 입력/편집
  • app.doc.search 문자열 찾기/바꾸기
  • app.doc.charShape 글자 모양 관리
  • app.doc.paragraph 문단 모양 관리
  • app.doc.tables 표 관리
  • app.doc.pages 쪽과 구역 관리
  • app.doc.objects 선택 개체와 그림 관리
  • app.doc.cursor 커서와 선택 영역 관리
  • app.doc.styles 스타일 관리
  • app.doc.references 책갈피, 하이퍼링크, 숨은 설명, 메모 이동 관리
app.file 문서 열기, 파일 내보내기, 파일 보안 작업 -
app.raw 한글 오토메이션 저수준 연동 방식 -
