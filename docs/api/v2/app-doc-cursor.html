app.doc.cursor

커서 이동, 현재 위치 저장/복원, 선택 영역을 관리합니다.

개요

TSHWPX API한글 오토메이션 연동
app.doc.cursor.move.*Move* Action 또는 MovePos
app.doc.cursor.position.*GetPosBySet, SetPosBySet
app.doc.cursor.selection.*Select*, MoveSel*, SelectText

이동 API

API설명연동
move.to(target, options)이름으로 정의된 위치로 이동합니다.MovePos
move.byId(moveId, options)MovePos ID로 직접 이동합니다.MovePos
move.document.begin(), end()문서 처음/끝으로 이동합니다.MoveDocBegin, MoveDocEnd
move.list.begin(), end()현재 리스트 처음/끝으로 이동합니다.MoveListBegin, MoveListEnd
move.line.begin(), end(), up(), down()줄 기준으로 이동합니다.MoveLine*
move.paragraph.*문단 처음/끝, 다음/이전 문단으로 이동합니다.MovePara*, MoveNextParaBegin
move.word.*낱말 처음/끝, 다음/이전 낱말로 이동합니다.MoveWord*, MoveNextWord
move.position.*논리 위치 단위로 앞뒤 이동합니다.MoveNextPos, MovePrevPos
move.char.*문자 단위로 앞뒤 이동합니다.MoveNextChar, MovePrevChar
move.direction.*화살표 방향으로 이동합니다.MoveLeft, MoveRight, MoveUp, MoveDown
move.page.*쪽 기준으로 이동합니다.MovePage*
move.view.*화면 기준으로 이동합니다.MoveView*
move.scroll.*스크롤 기준으로 이동합니다.MoveScroll*
move.section.*다음/이전 구역으로 이동합니다.MoveSectionDown, MoveSectionUp
move.column.*단 처음/끝, 다음/이전 단으로 이동합니다.MoveColumn*
move.cell.*표 셀 안에서 이동합니다.MovePos

위치 API

API설명연동
position.get()현재 커서 위치를 가져옵니다.GetPosBySet
position.set(position)저장한 커서 위치로 이동합니다.SetPosBySet

선택 API

API설명연동 Action
selection.start()선택을 시작합니다.Select
selection.all()전체 문서를 선택합니다.SelectAll
selection.column()칸 선택을 시작합니다.SelectColumn
selection.objectNext()다음 개체를 선택합니다.SelectCtrlFront
selection.objectPrevious()이전 개체를 선택합니다.SelectCtrlReverse
selection.cancel()선택 상태를 해제합니다.Cancel
selection.range(range)문자 범위를 직접 선택합니다.SelectText
selection.extend.*현재 선택 영역을 문서/줄/문단/낱말/방향 단위로 확장합니다.MoveSel*
