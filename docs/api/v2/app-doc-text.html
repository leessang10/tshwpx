app.doc.text

현재 활성 문서의 현재 커서 위치에서 텍스트를 입력하고 편집합니다.

개요

TSHWPX API한글 오토메이션 연동
app.doc.text.insert(text)Bridge insertText 호출
app.doc.text.lineBreak()BreakLine Action 실행
app.doc.text.paragraphBreak()BreakPara Action 실행
app.doc.text.tab()InsertTab Action 실행
app.doc.text.deletePreviousChar()DeleteBack Action 실행
app.doc.text.deleteNextChar()Delete Action 실행

하위 API

API설명연동
insert(text)현재 커서 위치에 문자열을 입력합니다.bridge.insertText(text)

paragraphBreak()현재 커서 위치에 문단 나누기를 입력합니다.BreakPara Action
tab()현재 커서 위치에 탭을 입력합니다.InsertTab Action
deletePreviousChar()커서 앞 글자를 삭제합니다.DeleteBack Action
deleteNextChar()커서 뒤 글자를 삭제합니다.Delete Action
