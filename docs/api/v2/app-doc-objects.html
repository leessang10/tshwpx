app.doc.objects

현재 선택된 개체 또는 그림을 대상으로 정렬, 순서, 이동, 크기, 그림 효과를 적용합니다.

주의

이 도우미들은 한글의 현재 선택 개체 또는 현재 선택 그림에 대해 동작합니다. 개체나 그림을 먼저 선택한 뒤 호출해야 합니다.

이 릴리스에는 파일 경로에서 새 그림을 직접 삽입하는 기능이나 새 도형을 생성하는 기능은 포함되지 않습니다. picture.change(options)는 선택된 그림의 파일을 바꾸는 용도입니다.
