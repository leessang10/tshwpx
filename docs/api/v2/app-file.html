app.file

한글 자동화 세션과 파일 시스템 사이의 입력, 내보내기, 파일 보안 작업을 관리합니다.

개요

TSHWPX API한글 오토메이션 연동
app.file.open(path, options)Bridge Open 호출
app.file.dialog.*, preview.open(), new()File* Action 실행
app.file.password.*, security.set, image.*, template.open파일 관련 Action + ParameterSet

위치

모듈의미
app실행 중인 한글 프로세스/자동화 세션
app.file그 세션에서 문서를 열거나 파일로 내보내는 작업
app.doc열려 있는 현재 문서의 내용/서식 작업

API설명연동

dialog.saveOptions()저장 옵션 대화상자를 엽니다.FileSaveOptionDlg
preview.open()미리 보기를 엽니다.FilePreview

password.change(options)문서 암호를 변경합니다.FilePasswordChange
password.readWrite.set(options)읽기/쓰기 암호를 설정합니다.FileRWPasswordNew
password.readWrite.change(options)읽기/쓰기 암호를 변경합니다.FileRWPasswordChange

image.save(options)문서를 이미지로 저장합니다.FileSaveAsImage
