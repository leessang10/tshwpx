app.file

한글 자동화 세션과 파일 시스템 사이의 입력, 내보내기, 파일 보안 작업을 관리합니다.

개요

TSHWPX API한글 오토메이션 연동
app.file.open(path, options)Bridge Open 호출
app.file.dialog.*, preview.open(), new()File* Action 실행
app.file.password.*, security.set, image.*, print.*, template.open파일 관련 Action + ParameterSet

위치

모듈의미
app실행 중인 한글 프로세스/자동화 세션
app.file그 세션에서 문서를 열거나 파일로 내보내는 작업
app.doc열려 있는 현재 문서의 내용/서식 작업

하위 API

API설명연동
open(path, options)문서를 엽니다.Bridge Open
new()새 문서를 만듭니다.FileNew
dialog.open()열기 대화상자를 엽니다.FileOpen
dialog.recent()최근 문서 열기를 실행합니다.FileOpenMRU
dialog.saveAs()다른 이름으로 저장 대화상자를 엽니다.FileSaveAs
dialog.saveOptions()저장 옵션 대화상자를 엽니다.FileSaveOptionDlg
preview.open()미리 보기를 엽니다.FilePreview
password.set(options)문서 암호를 설정합니다.FilePassword
password.change(options)문서 암호를 변경합니다.FilePasswordChange
password.readWrite.set(options)읽기/쓰기 암호를 설정합니다.FileRWPasswordNew
password.readWrite.change(options)읽기/쓰기 암호를 변경합니다.FileRWPasswordChange
security.set(options)배포/보안 설정을 적용합니다.FileSetSecurity
image.save(options)문서를 이미지로 저장합니다.FileSaveAsImage
image.options(options)이미지 저장 옵션을 설정합니다.FileSaveAsImageOption
print.print(options)현재 문서를 인쇄합니다.Print
print.setup(options)인쇄 설정을 적용합니다.PrintSetup
print.pdf(options)PDF 인쇄를 실행합니다.PrintToPDF
print.image(options)문서를 이미지로 출력합니다.PrintToImage
template.open(options)서식 파일을 엽니다.FileTemplate

print/export

app.file.print는 한글 인쇄 Action을 실행합니다. print.print, print.setup, print.pdf는 Print ParameterSet을 사용하고, print.image는 PrintToImage ParameterSet을 사용합니다.
