표 생성, 행/열 편집, 셀 병합과 분할을 관리합니다.

개요

하위 API

API설명연동 Action
insert(options)표를 생성합니다.TableCreate
rows.insertAbove(count)현재 행 위에 행을 삽입합니다.TableInsertUpperRow
rows.insertBelow(count)현재 행 아래에 행을 삽입합니다.TableInsertLowerRow
rows.append()표 끝에 행을 추가합니다.TableAppendRow
rows.delete()현재 행을 삭제합니다.TableDeleteRow
columns.insertLeft(count)현재 열 왼쪽에 열을 삽입합니다.TableInsertLeftColumn
columns.insertRight(count)현재 열 오른쪽에 열을 삽입합니다.TableInsertRightColumn
columns.delete()현재 열을 삭제합니다.TableDeleteColumn
cells.merge()선택한 셀을 병합합니다.TableMergeCell
cells.split(options)현재 셀을 여러 행/열로 나눕니다.TableSplitCell
cells.delete()현재 셀을 삭제합니다.TableDeleteCell
cells.distributeHeight()선택 셀의 높이를 균등하게 맞춥니다.TableDistributeCellHeight
cells.distributeWidth()선택 셀의 너비를 균등하게 맞춥니다.TableDistributeCellWidth

ParameterSet

APIParameterSet비고
insert(options)TableCreation표 생성 값
rows.insertAbove(count), rows.insertBelow(count)TableInsertLineCount 사용
columns.insertLeft(count), columns.insertRight(count)TableInsertLineCount 사용
rows.delete(), columns.delete()TableDeleteLineType으로 행/열 구분
cells.split(options)TableSplitCell분할 값
