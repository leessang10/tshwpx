app.doc

현재 활성 문서를 대상으로 하는 작업 진입점입니다.

개념

대상의미
app실행 중인 한글 프로세스/자동화 세션
app.file세션에서 문서를 열거나 파일로 내보내는 작업
app.doc그 세션의 현재 활성 문서 작업

API

API설명내부 동작
app.doc.save()현재 문서를 저장합니다.Bridge Save를 호출합니다.
app.doc.saveAs(path, format, arg)현재 문서를 다른 이름/형식으로 저장합니다.Bridge SaveAs를 호출합니다.
app.doc.close()현재 문서를 닫습니다. 한글 앱은 유지됩니다.Bridge Close를 호출합니다.
app.doc.text.insert(text)현재 커서 위치에 텍스트를 입력합니다.bridge.insertText를 호출합니다.
app.doc.charShape.set(options)글자 모양을 적용합니다.CharShape Action과 ParameterSet을 실행합니다.

모듈역할
